## Key v6 differences from v3

### Cursor pagination
```json
{"data": [...], "pagination": {"limit": 50, "nextCursor": "eyJ...", "hasMore": true}}
```
Cursor is a base64-encoded JSON of `{"lastId": N}`. Clients pass `?cursor=eyJ...&limit=50`.

### Error envelope (RFC 9457)
```json
{"type": "about:blank", "title": "Not Found", "status": 404, "detail": "No series with id 42", "instance": "/api/v6/series/42"}
```

### Field naming
Same as v3 for most fields (camelCase JSON) but without legacy cruft: no `tvRageId`, no `languageProfileId`, no `isNetCore`. Cleaner statistics nesting.

### errors.go

```go
// ProblemDetail implements RFC 9457.
type ProblemDetail struct {
    Type     string `json:"type"`
    Title    string `json:"title"`
    Status   int    `json:"status"`
    Detail   string `json:"detail,omitempty"`
    Instance string `json:"instance,omitempty"`
}

func WriteError(w http.ResponseWriter, r *http.Request, status int, detail string) {
    pd := ProblemDetail{
        Type:     "about:blank",
        Title:    http.StatusText(status),
        Status:   status,
        Detail:   detail,
        Instance: r.URL.Path,
    }
    w.Header().Set("Content-Type", "application/problem+json")
    w.WriteHeader(status)
    json.NewEncoder(w).Encode(pd)
}

func WriteNotFound(w http.ResponseWriter, r *http.Request, detail string) {
    WriteError(w, r, http.StatusNotFound, detail)
}
```

### pagination.go

```go
type Page[T any] struct {
    Data       []T        `json:"data"`
    Pagination Pagination `json:"pagination"`
}

type Pagination struct {
    Limit      int    `json:"limit"`
    NextCursor string `json:"nextCursor,omitempty"`
    HasMore    bool   `json:"hasMore"`
}

// DecodeCursor extracts lastId from a base64-encoded cursor string.
func DecodeCursor(cursor string) (lastID int64, err error)

// EncodeCursor creates a base64-encoded cursor from a lastId.
func EncodeCursor(lastID int64) string

// ParsePaginationParams reads limit and cursor from query params.
func ParsePaginationParams(r *http.Request) (limit int, lastID int64, err error)
```

# M20 — Backup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add automated and manual tar.gz database backups with configurable retention, accessible via v3/v6 API endpoints.

**Architecture:** A `backup.Service` in `internal/backup/` creates, lists, and deletes backup archives. The backup dir is `{config}/Backups/`. A scheduled task runs weekly. API endpoints expose backup management.

**Tech Stack:** Go stdlib (`archive/tar`, `compress/gzip`, `encoding/json`, `os`, `io`)

### Task 1: Backup Service

**Files:**
- Create: `internal/backup/backup.go`
- Create: `internal/backup/backup_test.go`

The Service needs:

```go
package backup

type Options struct {
    BackupDir string        // e.g., /config/Backups
    DBPath    string        // SQLite file path, empty for Postgres
    DBDialect string        // "sqlite" or "postgres"
    AppVersion string       // from buildinfo
    Retention  int          // max backups to keep (default 7)
    Log        *slog.Logger
}

type Manifest struct {
    Version    string    `json:"version"`
    DBDialect  string    `json:"databaseType"`
    Timestamp  time.Time `json:"timestamp"`
}

type Info struct {
    Name      string    `json:"name"`
    Size      int64     `json:"size"`
    Timestamp time.Time `json:"time"`
}

type Service struct { ... }

func New(opts Options) *Service
func (s *Service) Create(ctx context.Context) (Info, error)  // creates tar.gz, enforces retention
func (s *Service) List() ([]Info, error)                      // lists backups sorted newest first
func (s *Service) Delete(name string) error                   // deletes by filename
func (s *Service) FilePath(name string) string                // returns full path for download
```

**Create logic:**
1. Ensure backup dir exists (`os.MkdirAll`)
2. Generate filename: `sonarr2_backup_20060102150405.tar.gz`
3. Create tar.gz file
4. Write `manifest.json` entry
5. If SQLite and dbPath is non-empty, write `sonarr2.db` entry (copy file)
6. Close archive
7. Enforce retention (delete oldest if count > retention)
8. Return Info

**Tests:**
- `TestCreateBackup` — create with a temp SQLite file, verify tar.gz contents
- `TestListBackups` — create 2, verify both listed newest first
- `TestDeleteBackup` — create then delete, verify gone
- `TestRetention` — retention=2, create 3 backups, verify oldest removed
- `TestCreateBackupPostgres` — dialect=postgres, verify tar.gz has manifest only

func MountBackup(r chi.Router, svc *backup.Service) {
    r.Route("/api/v3/system/backup", func(r chi.Router) {
        r.Get("/", listBackups(svc))
        r.Post("/", createBackup(svc))
        r.Get("/{name}", downloadBackup(svc))
        r.Delete("/{name}", deleteBackup(svc))
    })
}
```

**V6** same pattern under `/api/v6/system/backup`.

The download handler uses `http.ServeFile` with the path from `svc.FilePath(name)`.

func extractSQLitePath(dsn string) string {
    path := strings.TrimPrefix(dsn, "file:")
    if idx := strings.Index(path, "?"); idx != -1 {
        path = path[:idx]
    }
    if path == ":memory:" || path == "" {
        return ""
    }
    return path
}
```
